Dom Tree Widget

Welcome to the Dom Tree Widget. This is a simple widget that allows you to view the structure of the DOM tree of the page. You can also view the styles of the elements and the event handlers attached to them. To get started, click on the Quick Start   link in the navigation menu. This widget is available for use on any rendered page.

Quick Start

To get started, you need to download the widget from the github repository. Link to repo is presented below. GitHub Page After downloading the widget, you need to install dependensies using the command npm i after that to make sure that everything was installed correctly you can run the command npm run dev and open the page in the browser. By default, the page will be available at the address http://localhost:8080
If you`re using Vanilla JS you need to make a bundle before using the widget. To do this, you need to run the command npm run build and after that you can import the widget in your project from the index.js file. To import the widget, you need to use the following code: import './path/to/bundled/index.js'; or the same for inserting the widget into the html file: <script src="./path/to/bundled/index.js"></script>
If you`re using React you can import the widget component using the following code: import { Widget } from './src/app/index.tsx';

available settings

The widget could be dragged and dropped to any place on the page. To do it you need to click on the icon and move it to the desired place. To hide the widget, you have to click on the icon
- To update the tree, you need to click on the icon
- You also have the opportunity to change the tree display option (hidden/expanded) by adding the corresponding flag at the top of the widget.
- To highlight the selected node, you can click on the node name in the list of nodes
